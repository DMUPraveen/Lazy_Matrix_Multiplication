# Lazy Multiplication
[![Open in molab](https://marimo.io/molab-shield.svg)](https://molab.marimo.io/github/DMUPraveen/Lazy_Matrix_Multiplication/blob/main/lazy_multiplication.py)

This is a repo dedicated to exploring the following Meme!

![A meme showing where matrix multiplication and digit concatenation yeilds the same result](<The_meme.jpg>)

We explore the questions such as,
- Are there more similar cases
- How many are there?
- How many where fist matrix is only single integer values?

The mathematical working out is given in main.pdf (Written using typst whose source code is in main.typ)
There is a marimo notebook used as the code base for the project.

If you are running it locally you can run it with (you need to have [uv installed](https://docs.astral.sh/uv/getting-started/installation/))

```{bash}
 uv run --with marimo marimo edit .\lazy_multiplication\
```

Alternatively you can run with mo-lab. [![Open in molab](https://marimo.io/molab-shield.svg)](https://molab.marimo.io/github/DMUPraveen/Lazy_Matrix_Multiplication/blob/main/lazy_multiplication.py)
